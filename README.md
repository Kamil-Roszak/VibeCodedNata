# Nata Catch

A mobile-friendly web game created for Nata Cola. Catch the falling bottles in your crate and achieve the highest score!

## Play Online

This game is designed to be hosted on GitHub Pages.

**To play:**
1. Go to the [Repository Settings](https://github.com/YOUR_USERNAME/REPO_NAME/settings/pages).
2. Under "Build and deployment", select **Source** as `Deploy from a branch`.
3. Select the `main` (or current) branch and the `/ (root)` folder.
4. Click **Save**.
5. The game will be available at `https://YOUR_USERNAME.github.io/REPO_NAME/`.

## How to Play

*   **Desktop:** Use the **Left** and **Right** arrow keys or **A** and **D** to move the crate.
*   **Mobile:** Touch and drag anywhere on the screen to move the crate.
*   **Goal:** Catch the Nata Cola bottles. Don't let them hit the ground!
*   **Game Over:** You have 3 lives. If you miss 3 bottles, the game ends.

## Features

*   **Nata Branding:** Custom graphics and colors matching the Nata visual identity.
*   **Persistence:** Your High Score is saved in your browser.
*   **Mobile Optimized:** Touch controls and responsive design.
*   **Audio:** Sound effects for gameplay events (can be muted).
